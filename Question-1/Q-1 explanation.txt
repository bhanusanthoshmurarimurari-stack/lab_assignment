Command: whoami && groups
Explanation: The 'whoami' command prints the active logged-in username, while the 'groups' command lists all security groups the user belongs to. This confirms that my account has elevated 'sudo' (administrator) privileges within the system environment.

Explanation: This command prints the value of the environment variable '$SHELL'. The output confirms that the system is currently utilizing the standard Bash (Bourne-Again Shell) command language interpreter for executing user inputs.

Command: pwd
Explanation: The 'pwd' (Print Working Directory) command displays the absolute file system path of the directory I am currently working in. This verifies that I am successfully running commands inside my designated lab workspace.

Command: ls -la
Explanation: The 'ls -la' command lists all contents within the current directory, using the '-l' flag for a detailed long-list format (permissions, sizes) and the '-a' flag to reveal hidden configuration files.
